# Kmeans-image-segmentation

This repository includes implementation of K-means for image segmentation. It uses RGB values along with the (x, y) location of
pixels and the texture value of each pixel for clustering.

# Dependencies
* numpy
* matplotlib
* PIL

# Testing
Run the following command to test the program :

`python main.py --input_path="test/test.jpg"`

Run `--help` to check all the arguments required :

`python main.py --help`
